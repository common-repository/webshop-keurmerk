== Description ==

**Over**

Deze plugin maakt het gemakkelijk voor webwinkels die aangesloten zijn bij het Webshop keurmerk om het logo/keurmerk
op hun Wordpress website te plaatsen. Plaats de widget bijvoorbeeld in de footer of header van uw website, voeg een link toe,
selecteer een grootte van het logo en klik op opslaan.



Alleen beschikbaar in het Nederlands.
Only available in Dutch.

== Installation ==

Upload het .zip bestand in de plugins map via /wp-content/plugins. Of upload het via uw Wordpress
website, ga naar plugins -> nieuwe plugin.

== Frequently Asked Questions ==

**Kan ik ook een andere link gebruiken dan de keurmerk site?**

Elke gewenste link kan worden gebruikt.

**De plugin werkt niet op mijn website?**

1. Controleer of de plugin geactiveerd is.
2. Controleer of je de laatste Wordpress versie hebt geinstalleerd.
